### 檔案路線結構:
    index        ( 首頁 )
    attractions  ( 人氣景點網美 )
    notification ( 通知訊息 )

    [user]
      memberView       ( 會員專區 )
        └ upgrade      ( 會員升級 )
      inviteFriends    ( 邀請好友加入 )
      settings         ( 個人資訊 )
        ├ profile      ( 顯示名稱及資訊 )
        ├ Instagram    ( Instagram 綁定 )
        ├ phoneNumber  ( 手機號碼 )
        ├ Email        ( Email )
        ├ password     ( 密碼 )
        ├ creditCard   ( 信用卡綁定 )
        └ remitAccount ( 匯款帳號 )
          └ newBank    ( 新增銀行 )
      hobbySetting     ( 興趣偏好設定 )
      withdrawal       ( 提領資訊 )
        └ newBank      ( 新增銀行 )
      customerService  ( 客服中心 )

    [invite]
      inviteMe              ( 邀約我的 - 邀約訊息 )
        ├ inviteMe-matched  ( 邀約我的 - 已成立 )
        ├ inviteMe-verified ( 邀約我的 - 已核銷 )
        └ inviteMe-invalid  ( 邀約我的 - 不成立 )
      meInvite              ( 我邀約的 - 邀約中 )
        ├ meInvite-matched  ( 我邀約的 - 已成立 )
        ├ meInvite-verified ( 我邀約的 - 已核銷 )
        └ meInvite-invalid  ( 我邀約的 - 不成立 )
